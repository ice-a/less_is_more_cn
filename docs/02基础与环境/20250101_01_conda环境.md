# Conda 环境管理

## 安装

### Linux

```bash
cd ~
wget https://github.com/conda-forge/miniforge/releases/download/24.3.0-0/Mambaforge-24.3.0-0-Linux-x86_64.sh
sh Mambaforge-24.3.0-0-Linux-x86_64.sh
source .bashrc
```

### Windows

下载 [Mambaforge-Windows-x86_64.exe](https://github.com/conda-forge/miniforge/releases/download/24.3.0-0/Mambaforge-Windows-x86_64.exe)，安装时全部勾选。

## 常用命令

```bash
# 版本与帮助
conda --version
conda --help

# 环境管理
conda env list                          # 查看所有环境
conda create -n py310 python=3.10       # 创建指定Python版本的环境
conda create -n myenv numpy scipy       # 创建包含指定包的环境
conda activate py310                    # 切换环境
conda deactivate                        # 退出当前环境
conda remove -n myenv --all             # 删除环境
conda create -n new_env --clone old_env # 复制环境

# 环境导入导出
conda env export > environment.yml      # 导出
conda env create -f environment.yml     # 导入

# 镜像加速
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```
