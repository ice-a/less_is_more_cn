## 一、 系统工具与维护

### [](https://linux.do/t/topic/803166#p-7364358-h-11-3)1.1 包管理器

- **UniGetUI** (免费, 推荐)
    - 说明：图形化的 Windows 包管理器前端，整合了 Winget、Scoop、Chocolatey 等多个流行的包管理工具，方便统一管理软件安装与更新。
    - 官方版下载：[Github 仓库](https://github.com/marticliment/UniGetUI)
- **Winget** (免费)
    - 说明：微软官方的包管理器，Windows 系统内置。
- **Scoop** (免费, 开源)
    - 说明：命令行包管理器，侧重于便携式（Portable）软件安装，通常不会污染系统环境变量。
    - 官方版下载：[官网](https://scoop.sh/)
- **Chocolatey** (免费/付费)
    - 说明：老牌的 Windows 包管理器，软件库庞大，更侧重于传统安装包形式的软件管理。
    - 官方版下载：[官网](https://chocolatey.org/)

### [](https://linux.do/t/topic/803166#p-7364358-h-12-4)1.2 系统优化与清理

- **WinUtil** (免费, 开源)
    - 说明：基于 PowerShell 的脚本工具集。提供程序安装、系统优化与去臃肿（Debloat）、配置管理、Windows 更新修复等功能。
    - 官方版下载：[Github 仓库](https://github.com/ChrisTitusTech/winutil)
- **Dism++** (免费)
    - 说明：强大的系统优化、清理维护工具。功能偏向系统深度优化（如系统备份还原、ESD转换等），需要一定的 Windows 系统知识。
    - 官方版下载：UniGetUI 有包；[Github 仓库](https://github.com/Chuyu-Team/Dism-Multi-language)
- **BleachBit** (免费, 开源)
    - 说明：轻量的系统清理工具。适合极客，提供命令行界面，适合自动化清理，不提供注册表清理功能。相比 CCleaner 更注重隐私擦除。
    - 官方版下载：UniGetUI 有包；[Github 仓库](https://github.com/bleachbit/bleachbit)
- **CCleaner** (免费/付费)
    - 说明：老牌系统清理工具。适合追求全面优化和简单操作的用户。提供注册表清理、启动项管理、软件卸载等功能（部分高级功能需付费）。近年来变得臃肿且有隐私争议，请谨慎使用其高级功能和注册表清理。
    - 官方版下载：UniGetUI 有包；[官网](https://www.ccleaner.com/)
    - 非官方便携版下载：[Download](https://www.aichunjing.com/soft/2018-07-06/229.html)
- **Mem Reduct** (免费)
    - 说明：轻量级的实时内存管理工具，用于监控和清理系统物理内存和虚拟内存。（Win11的内存管理已经较为完善，不推荐使用。）
    - 官方版下载：[官网](https://memreduct.org/mem-reduct-download/)

### [](https://linux.do/t/topic/803166#p-7364358-h-13-5)1.3 软件管理

- **Bulk Crap Uninstaller (BCUninstaller)** (免费, 开源)
    - 说明：一款批量程序卸载工具，能快速、彻底地卸载大量应用及其残留。它能检测到普通卸载工具无法发现的程序，非常适合清理系统。
    - 官方版下载：[官网](https://www.bcuninstaller.com/)；[Github 仓库](https://github.com/Klocman/Bulk-Crap-Uninstaller)
- **Geek Uninstaller** (免费)
    - 说明：深度卸载工具，能有效清理残留文件和注册表（注册表清理方式较激进，需谨慎）。界面简洁，响应快速。
    - 官方版下载：UniGetUI 有包；[官网](https://geekuninstaller.com/)
- **Revo Uninstaller** (免费/付费)
    - 说明：功能强大的卸载工具，提供“猎人模式”进行精准卸载。免费版功能有限，付费版提供更深度的残留清理和监控安装功能。
    - 官方版下载：UniGetUI 有包；[官网](https://www.revouninstaller.com/)
    - 非官方破解便携版：[Download](https://www.ghxi.com/revouninstaller.html)
- **Autoruns (Sysinternals Suite)** (免费)
    - 说明：Microsoft 官方出品的启动项管理工具。可以显示几乎所有自动启动的程序、服务、驱动程序、编解码器等。功能强大但操作有门槛，建议谨慎操作。
    - 官方版下载：UniGetUI 有包；[官网](https://learn.microsoft.com/zh-cn/sysinternals/downloads/autoruns)
- **ContextMenuManager** (免费, 开源，官方便携版)
    - 说明：专门用于管理和清理 Windows 系统右键菜单项。
    - 官方版下载：[Github 仓库](https://github.com/BluePointLilac/ContextMenuManager)。

#### [](https://linux.do/t/topic/803166#p-7364358-h-142-8)1.4.2 文件搜索

- **Everything** (免费)
    - 说明：基于文件名的最快速文件搜索工具，资源占用极低。仅搜索文件和文件夹名称。
    - 官方版下载：UniGetUI 有包；[官网](https://www.voidtools.com/)
- **Listary** (免费/付费)
    - 说明：文件搜索与快速启动工具。搜索速度快，支持搜索文件、文件夹、应用。提供收藏夹、快速跳转、命令执行等附加功能。
    - 官方版下载：UniGetUI 有包；[官网](https://www.listary.com/)
    - 非官方便携版：[Download](https://www.osssr.com/17639.html)
- **Fluent Search** (免费)
    - 说明：智能化的全局搜索工具。不仅搜索文件，还能搜索应用、进程、浏览器历史、应用内内容。支持屏幕搜索、文件预览和插件扩展，界面现代且高度可定制。
    - 官方版下载：UniGetUI 有包；[官网](https://fluentsearch.net/)

#### [](https://linux.do/t/topic/803166#p-7364358-h-143-9)1.4.3 磁盘空间分析

- **WizTree** (免费)
    - 说明：极速磁盘空间分析工具，通过读取 MFT（主文件表）快速定位大文件和文件夹占用情况。
    - 官方版下载：UniGetUI 有包；[官网](https://www.diskanalyzer.com/)
- **SpaceSniffer** (免费)
    - 说明：一款便携式磁盘空间分析工具，它通过树状图（Treemap）的形式直观地展示磁盘上文件和文件夹的占用情况，便于快速定位大文件。与 WizTree 相比，扫描速度稍慢，但界面展示更具特色。
    - 官方版下载：[官网](http://www.uderzo.it/main_products/space_sniffer/)

#### [](https://linux.do/t/topic/803166#p-7364358-h-144-10)1.4.4 文件处理工具

- **Bulk Rename Utility** (免费)
    - 说明：功能极为强大的批量重命名工具，支持正则表达式和复杂的重命名规则。
    - 官方版下载：UniGetUI 有包；[官网](https://www.bulkrenameutility.co.uk/)
- **dupeGuru** (免费, 开源)
    - 说明：免费开源的重复文件查找工具。它在查找音乐、图片等模糊匹配的重复文件方面有独到之处。
    - 官方版下载：UniGetUI 有包；[官网](https://dupeguru.voltaicideas.net/)
- **Duplicate Cleaner** (免费/付费)
    - 说明：dupeGuru 的商业替代品，专业的重复文件查找和删除工具，支持根据内容、名称、日期等多种方式比对。
    - 官方版下载：UniGetUI 有包；[官网](https://www.duplicatecleaner.com/)
    - 非官方破解便携版：[Download](https://www.osssr.com/6747.html)
- **WinMerge** (免费, 开源)
    - 说明：免费开源的文件和文件夹比较工具。对于常规的文本文件比较和合并已经足够，非常轻量。
    - 官方版下载：UniGetUI 有包；[官网](https://winmerge.org/)
- **Beyond Compare** (付费)
    - 说明：WinMerge 的专业替代品。支持文件内容、文件夹结构、FTP、云存储等多种数据源的比较、合并和同步。
    - 官方版下载：UniGetUI 有包；[官网](https://www.scootersoftware.com/download)
    - 破解版下载：(请自行搜索)

### [](https://linux.do/t/topic/803166#p-7364358-h-15-12)1.5 压缩与解压缩

- **NanaZip** (免费, 开源, 推荐)
    - 说明：基于 7-Zip 的现代化版本。专为 Windows 11 设计，完美支持新的右键菜单和 Mica 视觉效果。官方版下载：Microsoft Store；[Github 仓库](https://github.com/M2Team/NanaZip)
- **Bandizip** (免费/付费)
    - 说明：支持格式广泛（包括 ZIP, RAR, 7Z 等），兼容性强，解压速度快。免费版有广告，付费版（Professional）体验更佳。
    - 官方版下载：UniGetUI 有包；[官网](https://www.bandisoft.com/bandizip/)
    - 非官方破解版下载：[Download](https://www.ghxi.com/bandizip.html)
- **7-Zip** (免费, 开源)
    - 说明：拥有最高的压缩率（7z 格式），无广告。界面极为简洁（或称简陋），不支持创建 RAR 格式（仅支持解压）。
    - 官方版下载：UniGetUI 有包；[官网](https://www.7-zip.org/)
